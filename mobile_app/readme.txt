Welcome to iChat 1.0 Android
-------------------------------------

README for iChat 1.0 Android

Package contents
----------------

The iChat 1.0-android package comprises of the  following components:

1) source directory (iChat_androidApp/mobile_app/iChat/src/com/example/ichat/) - This contains the source code of iChat 1.0 for android.
2) LipiTK (iChat_androidApp/mobile_app/LipiTk/) - This contains the source code of written text detection.
Introduction
-------------------------------

iCHAT app works as an instant exchanging of texts. 
One of the main feature of app is that while texting it provide users with the two choices of mode
that is handwritting and keyboard. After detecting the changes application speaks the message.
